# Security Policy

## Supported versions

Security fixes are provided for the latest released minor version.

| Version | Supported |
| ------- | --------- |
| 0.1.x   | Yes       |
| < 0.1   | No        |

## Reporting a vulnerability

Use GitHub's private vulnerability reporting form:

<https://github.com/GagaKor/rtsp-backchannel/security/advisories/new>

Include the affected version, impact, reproduction steps, and any suggested
mitigation. Do not include camera credentials, stream URLs containing
credentials, or other secrets in a public issue.
